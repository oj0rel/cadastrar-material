Este sistema foi feito utilizando o método de sessionStorage do JavaScript.
Ele permite que o usuário cadastre informações de qualquer material que deseje, e exibe essas informações para ele.
Todas as informações que são inseridas passam por uma verificação, para que o sistema não permita a inserção de dados errados ou incompatíveis.
O sistema também possui funções de adicionar e retirar uma quantidade de material escolhida pelo usuário.
